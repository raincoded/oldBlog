# user

## 1.简要描述

- 根据id获取用户

##### 请求URL
- ` http://xx.com/api/user `
  
##### 请求方式
- GET 

##### 参数 params 传参
| 参数名 | 必选 | 类型 | 说明   | 默认值 |
| :----- | :--- | :--- | ------ | ------ |
| id     | 否   | int  | 用户id |        |

##### 返回示例

```js
{
    "code": 0,
    "msg": "获取成功！",
    "data": {
        "id": 23,
        "name": "金艳",
        "email": "c@qq.com"
    }
}
```

##### 返回参数说明 

| 参数名 | 类型   | 说明     |
| :----- | :----- | -------- |
| code   | int    | 请求正常 |
| msg    | string | 请求信息 |
| data   | object | 请求数据 |

##### 备注 

- 更多返回错误代码请看首页的错误代码描述
```js
{
    "code": 500,
    "msg": "用户不存在"
}
```
